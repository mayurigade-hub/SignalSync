# 🚦 Smart Traffic Signal Control (DRL vs Mathematical Model)

A simulation-based project that compares a **Mathematical Traffic Control Model** with a **Deep Reinforcement Learning (DRL) Agent** to optimize signal decisions at a 4‑way intersection.

## 🎯 Objective
Reduce traffic congestion by dynamically selecting the best traffic signal using prediction and adaptive decision-making.

## ⚙️ Key Components
- Traffic Simulation (Python + Pygame)
- Mathematical Weighted Scoring Model
- Deep Reinforcement Learning Agent
- Decision Comparison Module
- Performance Evaluation (waiting time, queue length)

## 🔄 Project Flow
Traffic Simulation → State Collection →  
Mathematical Decision → DRL Decision →  
Decision Comparison → Best Signal Applied

## 📊 Metrics Used
- Average Waiting Time
- Queue Length
- Vehicle Throughput

## 🛠 Tech Stack
- Python
- Pygame
- NumPy
- Reinforcement Learning
